# Parking-Management-System

About Project
This is a simple Parking Management System made in C++.
It is a beginner level project for practicing if else, switch case, loops, and basic input/output in C++.

Project Purpose
The purpose of this project is to simulate a small parking lot with 3 slots.
1.Users can park a car if there is an empty slot.
2.Users can remove a car from a slot.
3.Users can see which slots are empty or occupied.

How It Works
1.The program shows a menu with 4 options:
  1.Park Car
  2.Remove Car
  3.Show Parking Status
  4.Exit
2.User chooses an option by entering a number.
3.Program checks the slots and performs the action.
4.The menu repeats until the user chooses Exit.

Features
.Simple menu-driven program
.Uses variables to store parking slots
.Uses if else and switch case for decisions
.Uses while loop to repeat the menu
.Shows real-time parking status

How to Use
1.Open the project in any C++ IDE or compiler.
2.Run the program.
3.Follow the menu instructions to park or remove cars.
4.Check the parking status anytime using option 3.
5.Exit using option 4.
